## zshをインストール(apt)
```
$ sudo apt install zsh powerline fonts-powerline 
```
## デフォルトシェルをzshに変更
```
$ chsh -s /bin/zsh
```
## oh-my-zshをgitclone
```
$ git clone https://github.com/robbyrussell/oh-my-zsh.git ~/.oh-my-zsh 
```
## .zshrc にコピー
```
$ cp ~/.oh-my-zsh/templates/zshrc.zsh-template ~/.zshrc 
```
## .zshrcを実行
```
$ souce ./.zshrc
```
